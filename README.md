


### createi18n


`<button class="">
hello world
</button>`

becomes

`<button class="" i18n="@@FMW_FILENAME_HELLOWORLD">
hello world
</button>`

`<input placeholder="test">`

becomes 

`<input placeholder="test" i18n-placeholder="FMW_FILENAME_TEST">`




Run the script on an HTML page and see the magic.
